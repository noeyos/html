[a-z]
[A-Z]
[a-zA-Z]
[a-zA-Z0-9] 순서없음
[a-z][A-Z][0-9] 첫번째 글자는 소문자, 두번째 글자는 대문자, 세번째 글자는 숫자
[0-9] 0~9사이
[0-9] : 0~9사이
{3} : 3자리
{1,2} 1번 또는 2번
[0-9]{3}-[0-9]{4}-[0-9]{4}

+  : 한번 이상
*  : 0번 이상
?  : 0 또는 1
() : 그룹
.  : 모든 글자한글자를 대응
\. : '.' 자체 글자
\d : 숫자

[a-zA-Z]+[0-9]*@[a-zA-Z]+[0-9]*(\.[a-zA-Z]+){1,2}
